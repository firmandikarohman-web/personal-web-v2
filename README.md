# Rohman Firmandika - Personal Landing Page

A modern, professional personal website designed with an "Airy Professional" aesthetic. This landing page serves as a centralized hub for my portfolio, certifications, and daily activities.

## ✨ Key Features

- **Light Glassmorphism Design**: Minimalist and clean UI using modern CSS backdrop-filters and fluid layouts.
- **Dynamic Content Architecture**: All portfolio data (Certifications, Projects, Activities, and Reading List) is fetched dynamically from a single `data.json` file, making updates effortless.
- **Interactive Experience**:
  - **Magnetic Hover Effects**: Interactive response on icons and list items.
  - **Ambient Fluid Motions**: Floating organic shapes and parallax scroll effects.
  - **Reveal Animations**: Smooth staggered arrival of elements as you scroll.
- **Fully Responsive**: Seamlessly optimized for mobile, tablet, and desktop viewing.

## 🛠️ Tech Stack

- **Core**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Icons**: [Feather Icons](https://feathericons.com/)
- **Typography**: Inter (Google Fonts)
- **Data Management**: JSON Fetch API

## 🚀 Local Development

Since the project uses the Fetch API to load dynamic content, you MUST run this project through a local server to avoid CORS (Cross-Origin Resource Sharing) issues.

### Using VS Code (Recommended)
1. Install the **Live Server** extension.
2. Open `index.html`.
3. Click the **"Go Live"** button in the status bar or right-click and select **"Open with Live Server"**.

## 📂 Project Structure

- `index.html`: Main structure and bento-grid layout.
- `style.css`: Custom design system, glassmorphism tokens, and animations.
- `script.js`: Interactive logic and dynamic data rendering.
- `data/data.json`: Centralized content management.
